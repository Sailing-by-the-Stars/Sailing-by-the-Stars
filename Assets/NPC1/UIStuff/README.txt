Implementation
--------------------------------
Nameplate 
can be stretched without issue to fit the desired name
position black triangles at the sides as pictured in the blueprint UI image

Proceed icon
Only appears when dialogue can be skipped, optional is to have it slightly bob up and down/shrink and grow

Font
Instal font and set font size to 36 (at least that is what it is in illustrator), this for both the name plate and textbox, while the choice dialogue option is set to 32.

------------
Design
------------

Shape design mostly inspired by Animal Crossing, as that game focuses very much on dialogue and was the least boring thing I could find as reference games that were close to it in genre all had very boring design when it game to dialogue UI. I kept the design minimalistic and clean using basic shapes only to not distract from our dialogue which we want the player to focus on.

Symbolism is inspired by Polynesian culture (waves on the side our common tattoos there, and the black triangles have symbolistic significance in their culture).

Textbox color is parchment, like old maps would. Nameplate and dialogue option box are a blue tint to symbolize the sea and sailing. The dialogue option box is a pale blue to also give it a ghostly hue, very subtly hinting you might be a ghost. Giving this box a different tint will also make it more clear to the player that you are the one responding. The proceed icon is a yellow star that contrasts nicely with the blue, staying subtle and minimalistic.

Nunito is a go-to font for me game-wise, its very clean and easily readable. This font can also be easily customized if we want to go in a custom font direction.

Future iterations for this kind of dialogue UI, would be subtle animations for the text- and dialogue box, and a subtle wave effect that the UI would do when starting a conversation (like you are getting the conversation flowing.... funny)

Included are all assets, a blueprint to help set up the UI in unity and the font file for Nunito. If there are any more questions/feedback feel free to.
